# Société Générale España · IA 2026-2030

Informe web estático para comité de dirección sobre el impacto de la inteligencia artificial en Société Générale España entre 2026 y 2030.

## Archivos incluidos

- `index.html`: presentación completa e interactiva.
- `assets/societe-generale-logo.png`: logo usado en cabecera y favicon.
- `assets/fonts/`: fuentes `et-book` usadas por la web.
- `assets/sg-ai-briefing-2026-2030.mp4`: briefing audiovisual embebido en la columna derecha.
- `assets/sg-ai-briefing-poster.png`: imagen de portada del reproductor de vídeo.
- `.nojekyll`: evita que GitHub Pages procese el sitio con Jekyll.

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio en GitHub o abre uno existente.
2. Sube todos los archivos de este paquete a la raíz del repositorio.
3. En GitHub, entra en `Settings` > `Pages`.
4. En `Build and deployment`, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda. GitHub publicará la web en una URL del tipo:
   `https://TU-USUARIO.github.io/TU-REPOSITORIO/`
